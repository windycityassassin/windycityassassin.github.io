# windycityassassin.github.io

Personal portfolio. Static HTML, served by GitHub Pages at
[windycityassassin.github.io](https://windycityassassin.github.io/).

## Edit

Open `index.html` in any editor. Update the cards under `<section>`. Push to
`main`. Pages rebuilds in ~30 seconds.

## Add a project

Copy a `.card` div. Update title, tagline, description, tags, and links.
Add `featured` class for a live demo. Add `archived` class for repos you're
no longer running.
